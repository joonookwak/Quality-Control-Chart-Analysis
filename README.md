# Quality-Control-Chart-Analysis
# [Project] 통계적 공정 관리: 관리도 분석 (Control Chart Analysis)

본 프로젝트는 공정의 안정성을 모니터링하고 이상 원인을 감지하기 위한 **통계적 공정 관리(SPC)** 기법을 실습한 결과물입니다. 다양한 관리도를 통해 공정 데이터를 분석하고 해석하는 역량을 담고 있습니다.
## 👤 기여자
* **이름:** 곽준우 (Kwak Joonoo)
* **소속:** 고려대학교 (Korea University)
* **관심 분야:** Machine Learning, Computer Vision, Quality Analytics
## 🎯 분석 내용
* **관리도 선정:** 데이터의 특성에 맞는 관리도(X-bar, R, p, c 관리도 등) 선정 및 설계
* **공정 능력 분석:** 공정이 규격 내 제품을 생산할 수 있는 능력이 있는지(Cp, Cpk) 평가
* **이상 징후 포착:** 관리한계선(UCL, LCL) 이탈 및 런(Run) 등의 비정상 패턴 해석

## 🛠 주요 역량
* **이론적 배경:** 통계적 가설 검정 기반의 공정 관리 원리 이해
* **분석 도구:** Excel / Python / R (해당하는 도구를 적어주세요)
* **결과 해석:** 분석 결과를 바탕으로 한 공정 개선 제언
